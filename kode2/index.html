The Kode Network

Decentralized Video Transcoding
developed by Qencode

The problem

While video transcoding and other web services are growing rapidly, most customers are completely dependant on the terms and availability of just a few third-party providers.

High Centralization

Web services like transcoding are quickly being consolidated to smaller groups of providers, exposing the system to the risks inherent in all highly centralized service markets.

Price Discrimination

Providers give discounts to customers with high volume orders, making it difficult for smaller business to compete with larger corporations in their market.

Conflict of Interest

Many third-party providers also offer video and television services, competing directly with the customers who use their transcoding solutions and creating a conflict-of-interest that harms the customer.

The Solution

The Kode Network is a decentralized video transcoding platform that leverages the latest blockchain technologies trustless peer-to-peer solution that is cost-effective, available, secure and scalable.

Peer-to-Peer Network

The Kode Network is an infrastructure that supports collaboration across a large network of peers harnessing their idle processing power to request and execute transcoding tasks for each other.

Cost Effective

By using a fully autonomous algorithm to dynamically rent the idle processing power of competing nodes, the Kode Network will be able to achieve new levels of cost efficiency for transcoding services.

Trustless Consensus

By taking advantage of the latest blockchain technologies, verification methods, and incentive models, the interactions between participants on the network are always fair and secure.

Why Kode Network?

Efficiency

The founders of Qencode have already developed the most cost-efficient cloud video transcoding technology on the market, which will be leveraged to develop the Kode Network.

Functionality

Existing support for a wide range of formats, codecs, resolutions, features and transcoding functionality will be decentralized using blockchain technology over a peer-to-peer network.

UI and API Layer

The Kode Network will make it easy for anyone with limited or no blockchain experience to leverage the power of the network by providing abstraction layers to enable access to external customers.

Availability

Distribution over a network of peer-to-peer nodes reduces the risk of a single point of failure that could affect network performance as a whole.

Robust Video Transcoding

Using Kode Network, customets can convert videos into all the latest containers, formats and codecs, and create to videos with multiple quality and resolution settings.

Implementation Plan

phase 1
Internal Network

Existing Qencode infrastructure will be modified to develop a network of internal nodes running Kode Network software. Nodes will perform the functions required of each role, including distributed transcoding and the processing of payments using the Kode ERC-20 token on the Ethereum blockchain.

phase 2
Whitelisted Network

External nodes will be allowed to join the network via manual vetting process. This whitelisted peer-to-peer network will use a centralized verification system, as well a staking mechanism to incentivize good behavior.

Assessment of Technology

The Kode Network will choose one of two paths, depending on the ability of existing blockchain technologies to meet the scalability and cost requirements of the transcoding market.

option a
Ethereum & Truebit
phase 3a
Public Network with Truebit and Ethereum

The Kode Network will leverage Truebit, a protocol built on top of the Ethereum Blockchain, allowing any external node to fulfil key roles in the workflow.

phase 4a
Decentralized Public Network with Truebit and Ethereum

A fully decentralized peer-to-peer infrastructure using Truebit and Ethereum, that will allow any external node to perform any role in the workflow.

option b
Building Transcoding Blockchain
Phase 3B
Kode Network Blockchain

Software based on existing blockchain technology best suited to the requirements of a scalable and cost-efficient transcoding system will be developed. Anonymous public nodes will key roles new verification methods to ensure accurate resulted in a partly decentralized fashion.

Phase 4B
Decentralized Kode Network Blockchain

Kode Network will move to a fully decentralized transcoding blockchain, that maintains a high functionality infrastructure that supports trustless consensus while remaining cost-efficient and scalable.
