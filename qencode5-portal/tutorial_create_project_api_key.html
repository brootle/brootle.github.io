Create a Project

Project provide keys to access the Qencode API, as well as organize usage statistics.

Authenticate API

The API instance is used to communicate with the Qencode REST API. To authenticate, use your API key.

Copied
const apiKey = "5c924a5b76341"; // taken from your account
const qencodeApiClient = new QencodeApiClient(apiKey);
Create a Task

To begin, first create a task for your transcoding job.

Copied
let task = qencodeApiClient.CreateTask();
