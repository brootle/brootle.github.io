Setting up your project

Creating a Project

After logging into your account, click on the My Projects link in the header.

1.
To create Transcoding Profile, please make sure you are in the Edit Project section of your Project, and click + Add Transcoding Profile
2.
Enter a Profile Name for your Transcoding Profile. Pick something distinct which will make it easier to identify and manage in the future, for example: HLS at 375kbps (640x360).
3.
Select an Output Format from the dropdown.
To learn more about our output formats, review our Input and Output Formats section.
4.
Select an Video Codec from the dropdown.
To learn more about our output formats, review our Input and Output Formats section.
5.
Specify in the Output Path which folder and naming convention to use when saving this file to your storage or Transfer Method. An example would be videos/{uuid}.
Please note that for MP4 files, a separate Output Path must be specified for each resolution added.

Transcoding Profiles

You may create as many Transcoding Profiles as you like, one for each video file type tha you want outputted for transcoding job.

1.
To create Transcoding Profile, please make sure you are in the Edit Project section of your Project, and click + Add Transcoding Profile
2.
Enter a Profile Name for your Transcoding Profile. Pick something distinct which will make it easier to identify and manage in the future, for example: HLS at 375kbps (640x360).
3.
Select an Output Format from the dropdown.
To learn more about our output formats, review our Input and Output Formats section.
4.
Select an Video Codec from the dropdown.
To learn more about our output formats, review our Input and Output Formats section.
5.
Specify in the Output Path which folder and naming convention to use when saving this file to your storage or Transfer Method. An example would be videos/{uuid}.
Please note that for MP4 files, a separate Output Path must be specified for each resolution added.

Video Settings

Covers some of the main settings required to create the video portion of the transcoding files.

6.
Resolution. This allows you to add all the resolution you will need for this Transcoding Profile.
1.
To add a new resolution to this transcoding profile, click + Add Resolution
2.
Select the resolution you would like to add using the Output Resolution dropdown.
3.
Enter the bitrate for this video in the Bitrate (kbps) field.
4.
After you are done, click + Add Resolution to save.
You can continue adding resolutions until you have all that you need.

Audio Settings

7.
To edit Audio Settings, click the Audio Settings to expand the section. These setting modify the audio within the output video.
1.
Enter Max Audio Bitrate to specify the maximum bitrate for audio in kbps for output video.

Thumbnail Settings

8.
To edit Thumbnail Settings, click the Thumbnail Settings to expand the section.
1.
Click + Add Thumbnail to create a new thumbnail for this transcoding profile.
2.
Enter the percentage of the timeline from which you want to create the thumbnail in the Time (%) field. Click + Add Thumbnail. You can use 0 for the beginning, 100 for the end, and any integer in between.
3.
Click + Add Thumbnail Size to and enter enter the width and height of the thumbnail you want to create.
9.
After you've completed the setup you need, just click Save All and you're all set to start using your Transcoding Profile.
